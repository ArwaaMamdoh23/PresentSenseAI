# PresentSense AI

A mobile application that analyzes public speaking performance by evaluating speech, facial expressions, and body language using AI, and provides automated feedback.

Built with Flutter, Flask API, AWS, and AI models for speech and computer vision processing.

---

## Features

- Speech-to-text and grammar analysis (Whisper, Wav2Vec 2.0, T5)
- Body language and posture detection (MediaPipe, PoseNet)
- Facial emotion analysis (DeepFace)
- Automated performance scoring and feedback generation

---

## Tech Stack

- Flutter (Mobile App)
- Flask (Backend API)
- AWS (Cloud Deployment & Storage)
- Supabase (Database)
- AI Models: TensorFlow, OpenCV, NLP & Computer Vision

---

## How It Works

1. User uploads a presentation video  
2. Flutter app sends it to Flask backend  
3. Backend processes speech + video frames  
4. AI models analyze performance  
5. Feedback is returned to the app
